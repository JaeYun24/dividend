배당조회 Web Service 이용약관 변경안내

배당조회 Web Service를 이용해 주시는 여러분께 감사드립니다.
서비스의 제공 및 변경, 서비스의 중단, 회원 가입, 회원 탈퇴 및 자격 상실, 회원의 의무 등에 대해 보다 자세히 안내 드리기 위하여, 이용약관 내용이 2021년 4월 8일 자로 변경될 예정임을 안내 드립니다.

이용약관 변경사항
변경 전 변경 후

제4조 (서비스의 제공 및 변경)

1. [삼성전자 배당조회 Web Service]는 다음과 같은 업무를 수행합니다.
① 배당내역에 대한 정보조회 서비스 제공
② 기타 [삼성전자 배당조회 Web Service]가 정하는 업무

2. [삼성전자 배당조회 Web Service]는 회원이 서비스 이용 중 필요하다고 인정되는 다양한 정보에 대해서 전자메일이나 서신우편 등의 방법으로 회원에게 제공할 수 있으며, 회원은 원하지 않을 경우 가입신청 메뉴와 회원정보수정 메뉴에서 정보수신거부를 할 수 있습니다. 단, 서비스의 종료, 중단 기타 서비스 내용에 중대한 변경 등의 사유가 있을 경우에는 회원의 수신거부의사에도 불구하고 해당 내용을 제공할 수 있습니다.

제4조 (서비스의 제공 및 변경)

1. [삼성전자 배당조회 Web Service]는 다음과 같은 업무를 수행합니다.
① 삼성전자 주주에게 배당지급일로부터 1개월간 회원 가입과 등록 및 배당내역에 대한 정보조회 서비스 제공(배당지급일로부터 1개월이 경과한 이후에는 회원 가입과 등록 및 배당내역에 대한 정보조회가 불가능합니다)
② 기타 [삼성전자 배당조회 Web Service]가 정하는 업무
2. [삼성전자 배당조회 Web Service]를 통해 제공되는 모든 정보는 법적인 효력이 없으며, 재산 증명 등의 목적으로 사용될 수 없습니다.

3. [삼성전자 배당조회 Web Service]는 회원이 서비스 이용 중 필요하다고 인정되는 다양한 정보에 대해서 전자메일이나 서신우편 등의 방법으로 회원에게 제공할 수 있으며, 회원은 원하지 않을 경우 가입신청 메뉴와 회원정보수정 메뉴에서 정보수신거부를 할 수 있습니다. 단, 서비스의 종료, 중단 기타 서비스 내용에 중대한 변경 등의 사유가 있을 경우에는 회원의 수신거부의사에도 불구하고 해당 내용을 제공할 수 있습니다.

제 5조 (서비스의 중단)

1. [삼성전자 배당조회 Web Service]는 컴퓨터 등 정보통신설비의 보수 점검, 교체 및 고장, 통신의 두절 등의 사유가 발생한 경우에는 서비스의 제공을 일시적으로 중단할 수 있습니다.

2. 사업종목의 전환, 사업의 포기, 업체간의 통합 등의 이유로 서비스를 제공할 수 없게 되는 경우에는 [삼성전자 배당조회 Web Service]는 제 8조에 정한 방법으로 회원에게 통지할 수 있습니다.

제 5조 (서비스의 중단)

1. [삼성전자 배당조회 Web Service]는 컴퓨터 등 정보통신설비의 보수 점검, 교체 및 고장, 통신의 두절 등의 사유가 발생한 경우에는 서비스의 제공을 일시적으로 중단할 수 있습니다.
2. [삼성전자 배당조회 Web Service]는 제4조에서 규정한 바와 같이 배당지급일로부터 1개월간 회원 가입과 등록 및 배당내역에 대한 정보조회 서비스를 제공하며, 배당지급일로부터 1개월이 경과된 이후에는 다음 배당지급일까지 회원 가입과 등록 및 배당내역에 대한 정보조회 서비스는 중단됩니다. 단, 서비스 중단 기간에 삼성전자 주주의 별도 요청이 있는 경우 삼성전자는 [삼성전자 배당조회 Web Service] 이외의 방법으로 배당내역에 대한 정보를 제공할 수 있습니다.

3. 사업종목의 전환, 사업의 포기, 업체간의 통합 등의 이유로 서비스를 제공할 수 없게 되는 경우에는 [삼성전자 배당조회 Web Service]는 제 8조에 정한 방법으로 회원에게 통지할 수 있습니다.

제 6조 (회원 가입)

1. 삼성전자 주주로서 [삼성 주주 계정]에 등록된 회원은 [삼성전자 배당조회 Web Service]가 정한 가입 양식에 따라 회원정보를 기입한 후 이 약관에 동의한다는 의사표시를 함으로서 회원 가입을 신청합니다.

2. [삼성전자 배당조회 Web Service]는 제 1항과 같이 회원으로 가입할 것을 신청한 회원 중 다음 각 호에 해당하지 않는 한 회원으로 등록합니다.
① 가입 신청자가 이 약관 제 7조 제 3항에 의하여 이전에 회원자격을 상실한 적이 있는 경우. 다만 제 7조 제 3항에 의한 회원자격 상실 후 3년이 경과한 자로서 [삼성전자 배당조회 Web Service]의 회원 재가입 승낙을 얻은 경우에는 예외로 합니다.
② 모든 가입신청자는 실명확인 및 본인인증을 거쳐야 하나, 그러하지 아니한 경우
③ 등록 내용에 허위, 기재누락, 오기가 있는 경우.
④ 기타 회원으로 등록하는 것이 [삼성전자 배당조회 Web Service]의 기술상 현저히 지장이 있다고 판단되는 경우.

제 6조 (회원 가입)

1. 삼성전자 주주로서 [삼성 주주 계정]에 등록된 회원은 [삼성전자 배당조회 Web Service]가 정한 가입 양식에 따라 회원정보를 기입한 후 이 약관에 동의한다는 의사표시를 함으로서 회원 가입을 신청합니다. 각 배당지급일 기준 삼성전자 주주가 아닌 자는 회원 가입 신청을 할 수 없습니다.
2. [삼성전자 배당조회 Web Service]는 제 1항과 같이 회원으로 가입할 것을 신청한 회원 중 다음 각 호에 해당하지 않는 한 회원으로 등록합니다.
① 가입 신청자가 각 배당지급일 기준으로 삼성전자 주주가 아닌 경우
② 가입 신청자가 이 약관 제 7조 제 3항에 의하여 이전에 회원자격을 상실한 적이 있는 경우. 다만 제 7조 제 3항에 의한 회원자격 상실 후 3년이 경과한 자로서 [삼성전자 배당조회 Web Service]의 회원 재가입 승낙을 얻은 경우에는 예외로 합니다.
③ 모든 가입신청자는 실명확인 및 본인인증을 거쳐야 하나, 그러하지 아니한 경우
④ 등록 내용에 허위, 기재누락, 오기가 있는 경우.
⑤ 기타 회원으로 등록하는 것이 [삼성전자 배당조회 Web Service]의 기술상 현저히 지장이 있다고 판단되는 경우.

제 7조 (회원 탈퇴 및 자격 상실 등)

1. 회원은 [삼성전자 배당조회 Web Service]에 언제든지 탈퇴를 요청할 수 있으며 [삼성전자 배당조회 Web Service]는 즉시 회원 탈퇴를 처리합니다. 그리고 회원 탈퇴 시점부터 3개월 동안은 동일한 아이디로 회원가입을 할 수 없습니다.
2. 회원이 다음 각 호의 사유에 해당하는 경우 [삼성전자 배당조회 Web Service]는 회원자격을 제한 또는 정지 시킬 수 있습니다.
① 가입 신청 시에 허위 내용을 등록한 경우
② 다른 사람의 [삼성전자 배당조회 Web Service] 이용을 방해하거나 그 정보를 도용하는 등 정상적인 [삼성전자 배당조회 Web Service] 운영을 위협하는 경우
③ [삼성전자 배당조회 Web Service]에서 법령과 이 약관이 금지되거나 공서양속에 반하는 행위를 하는 경우
④ 삼성전자 주주임이 확인되지 않는 경우

3. [삼성전자 배당조회 Web Service]가 회원 자격을 제한·정지 시킨 후에도 전항 각호의 사유가 발생하여 [삼성전자 배당조회 Web Service]로부터 일정한 기간을 정하여 시정요청을 받았음에도 불구하고 그 기간 내 시정되지 아니하는 경우 [삼성전자 배당조회 Web Service]는 회원의 자격을 상실시킬 수 있습니다.

제 7조 (회원 탈퇴 및 자격 상실 등)

1. 회원은 [삼성전자 배당조회 Web Service]에 언제든지 탈퇴를 요청할 수 있으며 [삼성전자 배당조회 Web Service]는 즉시 회원 탈퇴를 처리합니다. 그리고 회원 탈퇴 시점부터 3개월 동안은 동일한 아이디로 회원가입을 할 수 없습니다.
2. 회원이 다음 각 호의 사유에 해당하는 경우 [삼성전자 배당조회 Web Service]는 회원자격을 제한 또는 정지할 수 있습니다.
① 가입 신청 시에 허위 내용을 등록한 경우
② 다른 사람의 [삼성전자 배당조회 Web Service] 이용을 방해하거나 그 정보를 도용하는 등 정상적인 [삼성전자 배당조회 Web Service] 운영을 위협하는 경우
③ [삼성전자 배당조회 Web Service]에서 법령과 이 약관이 금지되거나 공서양속에 반하는 행위를 하는 경우
④ 각 배당기준일을 기준으로 삼성전자 주주임이 확인되지 않는 경우

3. [삼성전자 배당조회 Web Service]가 회원 자격을 제한·정지시킨 후에도 전항 각호의 사유가 발생하여 [삼성전자 배당조회 Web Service]로부터 일정한 기간을 정하여 시정요청을 받았음에도 불구하고 그 기간 내 시정되지 아니하는 경우 [삼성전자 배당조회 Web Service]는 회원의 자격을 상실시키고 탈퇴 처리할 수 있습니다.

제 12 (회원의 의무)

1. 회원은 다음 행위를 하여서는 안됩니다.

① 신청 또는 변경 시 허위 내용의 등록
② 타인의 정보 도용
③ [삼성전자 배당조회 Web Service]에 게시된 정보의 변경
④ [삼성전자 배당조회 Web Service]가 정한 정보 이외의 정보 (컴퓨터 프로그램 등)의 송신 또는 게시
⑤ [삼성전자 배당조회 Web Service] 기타 제 3자의 저작권 등 지식재산권에 대한 침해
⑥ [삼성전자 배당조회 Web Service] 기타 제 3자의 명예를 손상시키거나 업무를 방해하는 행위
⑦ 외설, 욕설, 비속어 또는 폭력적인 메시지, 화상, 음성 기타 공서양속에 반하는 정보를 [삼성전자 배당조회 Web Service]에 공개 또는 게시하는 행위
⑧ 영리를 목적으로 하는 광고의 게시

제 12 (회원의 의무)

1. 회원은 다음 행위를 하여서는 아니 되며, 의무를 위반하는 경우 법적인 책임이 발생할 수 있습니다.

① 신청 또는 변경 시 허위 내용의 등록
